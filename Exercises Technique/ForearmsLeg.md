#### Dumbbell wrist curl

```bash
1. Коврик на пол, рука на скамье, ладонь выступает на дост. растоянии и см вверх
2. Максимальное опускание гантели до полувытянутых фалангий пальцев
3. Постепенное поднятие гантели за счет сокращения сгибателей запясьтя с закручиванием в ладонях
4. Медленное опускание с быстрым подконтрольным поднятием
```


#### Reverse Grip Barbell Biceps Curl

```bash
1. Возьмите штангу в руки обратным хватом, ладонями к себе.
2. Встаньте прямо, ноги на ширине плеч. Спина прямая, взгляд направлен вперед (без перекоса шеи вперед или назад).
3. Медленно поднимайте гриф до тех пор, пока кулак не окажется на уровне передней дельты.
4. Сделайте минимальную паузу и в более медленном темпе опустите гриф в исходную позицию.
5. Особенность движения заключается в том, что в отличие от классического варианта с прямым хватом, при обратных сгибаниях рук крайне важно полностью выпрямлять руки, то есть амплитуда включает полное разгибание локтя. В противном случае эффективность движения будет снижаться, также возрастет нагрузка на локтевой сустав.
```

#### Pronation Forearm Flex

```bash
1. Коврик на пол, рука на скамье, ладонь выступает на дост. растоянии и см вверх
2. Лямки зафиксированы на гантели, часть ленты хватается ладонью 4 пальцев
3. Поворот запястья по частовой стрелке на 5 - 10 град, после против на 120
4. При повороте запястья его нужно немного подвести к себе по принципу максимального ощущения нагрузки на пронатор
5. При повороте запястья выполняется подконтрольно, без дрожи и резкости
```

#### Leg extension machine

```bash
1. Сесть на установку, отрегулировать ур. сидения и прочего, икры под валик
2. Руки за рукоятки, на выдохе силой квадрицепса макс. поднять ноги
3. В макс. амплитуде 2 сек. и медленно опуститься
```

#### Bulgarian Split Splits

```bash
1. Станьте на небольшом расстоянии спиной от скамьи, возьмите в руки гантели.
2. Поставьте одну ногу назад, закинув на возвышенность, колено задней ноги не должно быть прямым.
3. Опорная стопа должна находиться под коленом. Руки с гантелями свободно опущены вдоль туловища по бокам.
4. На вдохе таз уходит вниз, колени сгибаются. Колено опорной ноги не должно выходить вперед за носок.
5. С выдохом разгибайте опорную ногу.
6. По окончании подхода снимите стопу задней ноги с возвышенности и поменяйте положение на другую сторону.
```
